// 01 — ABOUT

Background & Goals

Who I am, where I studied, and the kind of work I want to do.

My Background

I'm a MSc Cyber Security, Threat Intelligence & Cyber Forensics graduate from the University of Salford, Manchester (January 2024 – September 2025). My undergraduate degree was a BSc in Computer Science (Merit) from the University of Kerala, India (2020–2023), which gave me a solid foundation in networking, databases, and software development before I specialised in security.

My MSc dissertation focused on a real engineering problem: security gaps in AWS CI/CD pipelines. I built a live attack testbed using Terraform and GitHub Actions, deliberately introduced misconfigurations, and ran three MITRE ATT&CK-mapped attacks. A key finding was that GuardDuty raised zero alerts on pipeline abuse despite full CloudTrail logging — a genuine detection gap with real implications for organisations relying on AWS-native tooling alone.

My second major project tackled S3 and EC2 misconfiguration security — I replicated the Capital One 2019 breach technique by extracting IAM credentials via unauthenticated IMDSv1 metadata queries on a misconfigured EC2 instance. These aren't academic exercises; they're the attack paths real organisations leave open.

Outside of security research, I've been working at Amazon UK since March 2025 as a Sortation Associate, managing 15–25 delivery partners per shift and maintaining 99%+ accuracy on exception resolution. I was recognised with the Amazon in the Community Award (September 2025). Before that, I completed a Software Testing internship at Luminar Technolab in India, where I found and documented 50+ bugs using Python scripts and SQL queries.

Career Goals

I'm targeting roles in Cloud Security Engineering, Security Operations, or DevSecOps where I can apply practical AWS knowledge and Python automation to protect infrastructure at scale. I want to work somewhere that treats security as an engineering discipline — where threat modelling, automated detection, and secure-by-default design are standard practice. Available full-time from September 2026, with the right to work in the UK on a Graduate Visa until January 2028.
